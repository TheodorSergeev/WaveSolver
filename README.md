# WaveSolver

Solving the wave equation for the Dirichlet, Neumann, Mur and PML boundary conditions.

## Getting Started

### Prerequisites

```
Gnuplot (ver. 5.2 or higher)
CMake (ver. 2.8 or higher)
ffmpeg (ver 4.1 or higher)
```

### Building

```
mkdir build
cd build
cmake ../
make
```

### Running

```
./WaveSolver
```

### Plot and animate

```
../plot.sh
```
